# 🐾 Buddy - Claude 宠物系统

一个集虚拟宠物、CLI伙伴和宠物管理于一体的 Python 宠物系统。

## 功能

### 🎮 虚拟宠物 (电子宠物)
- 6种宠物类型: 猫、狗、兔子、龙、狐狸、熊猫
- 喂食系统 (6种食物)
- 游戏系统 (5种游戏)
- 成长进化系统 (蛋→幼年→少年→青年→成年)
- 心情/饱腹度/体力/健康状态条
- 等级经验系统 + 金币系统
- 存档/读档

### 🖥️ CLI 伙伴 (编程陪伴)
- 6种 ASCII 艺术宠物 (带多种表情状态)
- 编程鼓励语
- 关键词互动 (识别情绪并回应)
- 随机事件系统

### 📋 宠物管理 (真实宠物)
- 宠物档案管理 (种类、品种、年龄、体重)
- 疫苗记录 + 提醒
- 医疗记录
- 喂食计划
- 备注系统

## 快速开始

```bash
python -m buddy.main
```

## 项目结构

```
buddy/
├── __init__.py          # 包初始化
├── main.py              # 主入口 (交互菜单)
├── virtual_pet/
│   ├── __init__.py
│   └── pet.py           # 虚拟宠物核心逻辑
├── cli_companion/
│   ├── __init__.py
│   └── companion.py     # CLI 伙伴 (ASCII 艺术)
├── pet_manager/
│   ├── __init__.py
│   └── manager.py       # 真实宠物管理
└── data/                # 存档数据目录
```

## 需求

- Python 3.10+
- 无需额外依赖，纯标准库实现
