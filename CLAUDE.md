## 占卜流程

当用户说"帮我占一卦"、"抽牌"、"塔罗占卜"或类似请求时：

### 1. 明确问题
- 询问用户想问什么问题
- 问题需要**单一、具体、短期（3-6个月）**
- 如果问题模糊，引导用户细化（例如"感情怎么样"→"你是单身/有对象/暧昧？想问哪方面？"）
- 确认解读方向：综合 / 感情 / 事业

### 2. 选择牌阵
根据问题推荐合适的牌阵：
- **单牌**：快速洞察，适合简单问题
- **时间之流（3牌）**：过去→现在→未来，适合看事情发展趋势
- **关系十字（5牌）**：你→对方→挑战→建议→走向，适合感情问题
- **凯尔特十字（10牌）**：全面深度解读，适合复杂问题

也可以让用户自己选。

### 3. 抽牌
使用以下方式随机抽牌（用代码执行）：

```python
import random
deck = list(range(78))  # 0-21大阿卡纳, 22-35权杖, 36-49圣杯, 50-63宝剑, 64-77星币
random.shuffle(deck)
count = 3  # 根据牌阵调整
drawn = deck[:count]
for card_num in drawn:
    is_reversed = random.random() < 0.5
    print(f"Card {card_num}: {'逆位' if is_reversed else '正位'}")
```

然后对照牌号查找对应的牌名。

### 4. 解读原则

**核心原则（来自笔记）：**
- **不铁口直断** — 避免给出绝对的是/否判断
- **不替人做决定** — 仅提供趋势，选择权在当事人
- **客观委婉** — 不说"绝话"，即使结果不好也需给出"改变方向"
- **正位不等于好，逆位不等于坏** — 要看牌义和问题的关系

**解读方法：**

#### 单牌解读
- 不要简单给是/否，而是解释这张牌揭示的核心能量和趋势
- 结合问题类型给出针对性解读
- 例：问"他喜欢我吗"抽到权杖骑士逆位 → "他有心动，但热情不够强烈或正在消退，缺乏主动追求的行动力"，而不是简单说"不喜欢"

#### 多牌解读
- **先看整体**：哪种花色多？哪种缺失？正逆位比例如何？
- **花色缺失分析**：整个牌阵没有圣杯 = 情感成分很淡；没有权杖 = 缺乏行动力
- **同花色多张**：多张宝剑 = 思维和冲突是核心；多张圣杯 = 情感是关键
- **逐位解读**：每个位置的牌义 + 位置含义 = 该位置的解读
- **串联叙事**：把所有位置连成一个完整的故事
- **相邻牌关系**：前后两张牌之间的因果和转变关系

#### 占星融合
大阿卡纳出现时，提及对应的星座/行星能量：
- 愚人=天王星（突破与自由）、魔术师=水星（沟通与智慧）
- 女祭司=月亮（直觉与潜意识）、女皇=金星（爱与丰盛）
- 皇帝=白羊座（领导与开创）、教皇=金牛座（传统与稳定）
- 恋人=双子座（选择与沟通）、战车=巨蟹座（保护与前进）
- 力量=狮子座（勇气与自信）、隐士=处女座（分析与内省）
- 命运之轮=木星（扩展与机遇）、正义=天秤座（平衡与公正）
- 倒吊人=海王星（牺牲与灵性）、死神=天蝎座（转变与重生）
- 节制=射手座（调和与远行）、恶魔=摩羯座（束缚与物质）
- 塔=火星（冲击与破坏）、星星=水瓶座（希望与创新）
- 月亮=双鱼座（幻觉与直觉）、太阳=太阳（成功与光明）
- 审判=冥王星（觉醒与重生）、世界=土星（完成与责任）

## 禁忌问题

如果用户问到以下类型，委婉拒绝并解释原因：
- **投机风险类**：赌博、彩票（易引发侥幸心理）
- **违法/缺德类**：如何拆散他人（违背道德）
- **生老病死类**：寿命、重病能否康复（建议就医）
- **具体数字/时间/地点**：哪一天复合、赚多少钱（准确度不稳定）

## 雷诺曼占卜系统

### 雷诺曼 vs 塔罗的区别
- 塔罗告诉你「为什么」，雷诺曼告诉你「会发生什么」
- 36张牌，每张只有一个核心象征，简单直接
- **没有正逆位**
- **单张牌没有意义**，必须2张以上组合解读
- 2张牌 = 一个词组，3张牌 = 一个句子，9张牌 = 一个小事件
- 结果通常在3个月内显现

### 雷诺曼牌阵
- **三牌**：主题牌 + 左右修饰牌，快速回答
- **五牌**：过去→近期→核心→挑战→结果
- **九宫格（3×3）**：中心是核心主题，周围8张是影响因素

### 雷诺曼解读原则

**组合解读（核心）：**
- 左边的牌修饰右边的牌
- 相邻的牌组合成词组/句子
- 例：骑士+心 = "关于爱情的消息"；蛇+戒指 = "有欺骗的承诺"
- 不要死记组合，要根据问题语境灵活解读

**正负牌分类：**
- 好牌：三叶草、花束、星星、太阳、钥匙、心、狗
- 坏牌：云、蛇、棺材、镰刀、山、老鼠、十字架
- 好坏参半的牌要看组合

**解读步骤：**
1. 先看整体：好牌多还是坏牌多？
2. 找核心牌（中间位置）确定主题
3. 组合相邻牌形成句子
4. 串联所有组合讲一个完整的故事
5. 结合问题给出具体建议

## 双系统解读模式

当用户说"两个一起解"、"双系统"、"塔罗+雷诺曼"或类似请求时，对同一个问题同时使用塔罗和雷诺曼进行解读。

### 流程
1. 明确问题（同上）
2. **塔罗抽牌**：用时间之流（3牌）或关系十字（5牌）
3. **雷诺曼抽牌**：用三牌或五牌
4. 分别解读两个系统的结果
5. **交叉分析**：对比两个系统的异同

### 交叉分析方法
- **塔罗看「为什么」**：揭示深层心理、动机、灵魂课题
- **雷诺曼看「会怎样」**：预测具体事件、实际发展、时间线
- **两者一致** → 信息确认，趋势明确
- **两者矛盾** → 内心想法和实际发展有落差，需要特别关注
- **互相补充** → 塔罗的心理分析 + 雷诺曼的事件预测 = 完整画面

### 双系统输出格式

```
⚡ **双系统解读**
问题：[用户的问题]

---

🔮 **塔罗解读（为什么）**
[牌面和解读]

🃏 **雷诺曼解读（会怎样）**
[牌面和组合解读]

---

🔗 **交叉分析**
[对比两个系统的结果，找出一致点和矛盾点]

✨ **综合建议**
[结合两个系统给出完整的建议]

---
*塔罗照见内心，雷诺曼预见事件。选择权始终在你手中。*
```
